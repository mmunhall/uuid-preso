UUID Versions

xxxxxxxx-xxxx-Mxxx-Nxxx-xxxxxxxxxxxx

  1. MAC address & datetime
  2. DCE security
  3. MD5 hash & namespace
  4. Random
  5. SHA1 hash & namepsace
  • Most important to remember that ANY version will work for our purposes, but the higher the version the better.
    1. MAC address and number of 100 nanosecond intervals since the adoption of the Gregorian calendar. This version is not "sufficiently opaque".
    2. Version 1 with some bytes replaced with a POSIX UID or other identifier
    3. "Named UUID": MD5 hash from a URL, FQDN, some other object
    4. "Random UUID": Derived from random numbers
    5. Same as version 3, but uses SHA-1 because MD5 has been compromised
