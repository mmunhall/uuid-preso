java.util.UUID

  • fromString(String name): Create an instance from a string
  • nameUUIDFromBytes(byte[] name): Create a named instance
  • randomUUID(): Give me a UUID, please.
  • toString()
  • fromString(): Must be formatted with dashes, which oracle does not provide. Not case sensitive.
  • nameUUIDFromByotes(): Useful when MD5 hashing a string
  • randomUUID(): Version 4
  • toString(): Returns the canonical representation of a UUID, but with dashes. Not suitable for Oracle's hextoraw()!
