UUID Variants and Versions

xxxxxxxx-xxxx-Mxxx-Nxxx-xxxxxxxxxxxx

Variants

  • Spec defines only one variant
  • Format: 8-4-4-4-12
  • N indicates the variant (8, 9, a, or b)
  • M indicates the version (3, 4, or 5)

Versions

  1. Mac address & date-time
  2. DCE Security
  3. MD5 hash & namespace
  4. Random
  5. SHA-1 hash & namepsace
  • Variant is the format of the UUID
  • There are other variants, but included only for backward compatibility.
